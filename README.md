# annotate-html

A web service that annotates HTML snippets by hyperlinking names.

## Technologies used

- Node v12
- Express.js
- [cheerio](https://cheerio.js.org/)

## How to run

1. Install Node v12

```
nvm use 12
```

To install `nvm`, please refer to [this page](https://github.com/nvm-sh/nvm)

2. Install npm packages

```
npm i
```

3. Run server

```
npm start
```

## How to test

```
npm test
```
